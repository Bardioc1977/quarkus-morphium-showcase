Aggregation Pipeline Example

$project → $group → $sort → $limit 
Aggregator<SalesRecord, Map> agg = morphium.createAggregator(SalesRecord.class, Map.class);
agg.group(Map.of(
    "_id", "$region",
    "totalRevenue", Map.of("$sum", Map.of("$multiply", List.of("$unit_price", "$quantity"))),
    "totalOrders", Map.of("$sum", 1)
));
agg.sort(Map.of("totalRevenue", -1));

Service Code (Morphium API)

// Aggregation: Sales by Region
Aggregator<SalesRecord, Map> agg = morphium.createAggregator(
    SalesRecord.class, Map.class);
agg.group("$region")
    .sum("totalRevenue", Map.of("$multiply",
        List.of("$unit_price", "$quantity")))
    .sum("totalOrders", 1)
    .avg("avgOrderValue", ...)
    .end();
agg.sort(Map.of("totalRevenue", -1));
agg.aggregate();

// Project with date extraction
agg.project(Map.of(
    "month", Map.of("$month", "$sale_date"),
    "revenue", Map.of("$multiply",
        List.of("$unit_price", "$quantity"))));

// Distinct values
morphium.createQueryFor(SalesRecord.class)
    .distinct("region");
