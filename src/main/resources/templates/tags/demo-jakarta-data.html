Repository Hierarchy

The showcase uses all three repository levels:

RepositoryExtendsExtra Features
BlogPostRepository BasicRepository save, findById, delete, findAll
ProductRepository MorphiumRepository + insert, update + distinct(), morphium(), query()
EmployeeRepository MorphiumRepository + insert, update + distinct(), morphium(), query()

MorphiumRepository extends CrudRepository with three Morphium-specific methods that have no equivalent in Jakarta Data 1.0:

  • distinct("field") — unique values across all documents
  • morphium() — escape hatch to the full Morphium API (aggregation, atomic updates, etc.)
  • query() — typed Morphium Query for complex conditions
