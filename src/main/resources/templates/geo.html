Store Locator

Geospatial queries with 2dsphere indexes, $near and $geoWithin operations

Morphium Features Demonstrated

GeoJSON Point 2dsphere Index $near Query $geoWithin @Entity @Index

Entity Definition

@Entity(collectionName = "stores")
public class Store {
    @Id private MorphiumId id;
    @Index private String name;
    private double[] location; // [lng, lat] for 2dsphere
    private List<String> services;
}

Service Code (Morphium API)

// Ensure 2dsphere index on startup
morphium.ensureIndicesFor(Store.class);

// Store with GeoJSON coordinates
Store store = Store.builder()
    .name("Munich Store")
    .location(new double[]{11.576, 48.137})
    .build();
morphium.store(store);

// Batch store multiple locations
morphium.storeList(stores);
