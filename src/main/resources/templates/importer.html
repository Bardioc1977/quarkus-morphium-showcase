Morphium Features Demonstrated

storeList (Batch) @WriteBuffer(500, WRITE_NEW) SequenceGenerator push / pull (Array) set (Atomic) unset (Field Remove) MorphiumIterator

WriteBuffer + Sequence Code

@Entity(collectionName = "import_records")
@WriteBuffer(size = 500, strategy = WRITE_NEW, timeout = 5000)
public class ImportRecord { ... }

// Sequence Generator for auto-increment IDs
SequenceGenerator seq = new SequenceGenerator(morphium, "import", 1, 1);
record.setImportNumber(seq.getNextValue());
morphium.storeList(records); // batch insert

// Array operations
morphium.push(query, "tags", "processed");   // add to array
morphium.pull(query, "tags", "pending");     // remove from array
morphium.unset(query, "source");               // remove field

Service Code (Morphium API)

// SequenceGenerator for auto-increment IDs
SequenceGenerator seq = new SequenceGenerator(
    morphium, "import_number", 1, 1);
record.setImportNumber(seq.getNextValue());

// Batch insert
morphium.storeList(records);

// Array operations
morphium.push(query, ImportRecord.Fields.tags, "new-tag");
morphium.pull(query, ImportRecord.Fields.tags, "old-tag");

// Atomic field update
query.set(ImportRecord.Fields.status, "PROCESSED",
    false, false, null);

// Remove a field entirely
query.unset(ImportRecord.Fields.source);
